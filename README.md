# 🍔 Food Delivery Data Analysis

## 📌 Project Description

This project analyzes food delivery data using Python and Excel.
It helps to find average delivery time, popular food items, and peak order hours.

## 📊 Features

* Calculate average delivery time
* Identify most popular food item
* Analyze peak hours
* Show results using graphs

## 🛠 Tools Used

* Python (Pandas, Matplotlib)
* Microsoft Excel

## ▶ How to Run the Project

1. Install Python
2. Install required libraries:
   pip install pandas matplotlib
3. Run the program:
   python analysis.py

## 📈 Output

* Bar chart for popular food
* Pie chart for peak hours
* Delivery time analysis
